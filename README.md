# Bank Management System

## Overview
This is a simple Java-based console application designed to manage basic banking operations. This project provides a good platform for a new developer to understand fundamental Object-Oriented Programming (OOP) concepts and file handling (if used).

## Features

The system supports the following key features:
* **Account Creation:** Opening bank accounts for new customers.
* **Balance Inquiry:** Checking the current balance in an account.
* **Deposit Funds:** Depositing money into an account.
* **Withdrawal:** Withdrawing money from an account.
* **Exit:** Shutting down the application.

## Technology Stack
* **Language:** Java (Core Java)
* **Environment:** Eclipse IDE
* **Version Control:** Git / GitHub

## How to Run

Follow these steps to run this project on your local system:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/israil-mp3/Bank-Account-Management-.git](https://github.com/israil-mp3/Bank-Account-Management-.git)
    ```

2.  **Import to Eclipse:**
    * Open Eclipse.
    * Select **File > Import > Git > Projects from Git**.
    * Choose **Existing local repository** and select your project before clicking **Finish**.

3.  **Run the Application:**
    * Open the `Main.java` file (select the file if the name is different).
    * Right-Click on the file and select **Run As > Java Application**.
